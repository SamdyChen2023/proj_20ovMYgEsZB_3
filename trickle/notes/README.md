# Fancy Landing Page

A modern, responsive landing page built with React and Tailwind CSS featuring a gradient design, smooth animations, and engaging user interface.

## Features

- **Modern Design**: Gradient backgrounds with glass-morphism effects
- **Responsive Layout**: Fully responsive across all devices
- **Smooth Animations**: Engaging hover effects and transitions
- **Component-Based**: Modular React components for easy maintenance
- **Performance Optimized**: Fast loading and smooth scrolling

## Structure

### Landing Page (index.html)
- **Header**: Fixed navigation with scroll-based styling
- **Hero**: Eye-catching hero section with gradient background
- **Features**: Grid layout showcasing key features with icons
- **Stats**: Statistics section highlighting key metrics
- **CTA**: Call-to-action section to drive conversions
- **Footer**: Complete footer with links and social media

### About Page (about.html)
- **About Hero**: Gradient hero section with mission statement
- **Story**: Company history and journey narrative
- **Values**: Core company values with icons
- **Team**: Team member profiles with photos

## Design System

- **Primary Color**: Indigo (#6366f1)
- **Secondary Color**: Amber (#f59e0b)
- **Typography**: System fonts for optimal readability
- **Components**: Reusable button styles and feature cards

## Technologies

- React 18
- Tailwind CSS
- Lucide Icons
